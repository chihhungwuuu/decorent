# Decorent - 線上購物網站
這是一個家具線上購物網站。採用前後端分離開發模式，並透過RESTful API傳遞資料。

## 使用技術
- Spring Boot 3.3.0
- Java 17
- Angular 17
- MySQL 8.0
- Spring Data JPA
- Spring Data REST
- Spring Security
- JWT
